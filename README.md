# Data Structure 

This repository contains my implementations and notes while studying **Data Structures using C++**.

The goal of this project is to deeply understand how fundamental data structures work by **implementing them from scratch**, rather than simply using standard library implementations.

Through this process, I focus on strengthening my **algorithmic thinking, problem-solving skills, and low-level understanding of memory and performance**.

---

## 📚 Topics Covered

The repository is organized by topic. Each folder contains implementations and small experiments related to a specific data structure or concept.

Examples include:

* Swap / Basic C++ Memory Concepts
* Dynamic Arrays (Vector implementation)
* Linked List
* Stack
* Queue
* Tree
* Graph
* Sorting Algorithms
* Searching Algorithms

More topics will be added as the study progresses.
